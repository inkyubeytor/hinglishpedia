Aksar log Smartphone ke hang hone ki problem se preshan rahte hain. Oos waqt aap phone ko restart karte hain ya phir wait karte hain ki yeh problem kuch hi der mein theek ho jayegi . Ek baar phone ke hang hone ki problem shuru ho gyi toh yeh aksar preshan karti rahti hain. Lekin is problem ka samadhaan hain aur aap asaani se apne phone ko theek kar sakte hain. Is post mein hum kuch aise tips lekar aaye hain jisse aap apne smartphone ke hang hone ki problem se nijaat pa sakte hain.  • Task Manager :- Phone ke istemal ke dauran hum kai saari application ek saath kholte jate hain. Sabhi application background mein run kar rahe hote hain. Aise mein koshish kare ki task manager ke jariye application ko band karde. Dher saari application ke run hone ki wajah se bhi phone hang ho sakta hain.  • Cloud Storage :- Android smartphone mein data ko store karne ke liye cloud storage behatar option hain. Aise folder ya file jinka istemal aap behad kam karte hain , unhe cloud storage par store kar sakte hain. Isse phone ki internal memory khali hogi. Lekin is baat ka dhyaan rakhe ki cloud storage ko upyog karne ke liye internet connection ka hona jaroori hain. Google Drive, One Drive aur Drop Box aadi Cloud storage application hain.  • Phone memory ko kare khali :- Android phone ke upyog ke dauran aksar anchaha data cache memory mein install ho jata hain. Android phone mein browser application aur game samet kisi bhi feature ka upyog karne ke dauran kuch unchaha data cache memory mein store ho jata hain. Yeh data jaise jaise bharne lagta hain vaise vaise phone dheema hona lagta hain aur hang ki problem shuru ho jati hain. Aise mein aap cheche cache memory ko delete kar phone ke hang hone ki problem se nijaat paa sakte hain. Cache memory ka option aapko setting mein jakar storage mein milega.   • Factory data reset :- agar aapko in cheezo se bhi phone ke hang hone ki samsya ka samadhaan nahi ho raha hain toh aap factory data reset kar sakte hain. Lekin Factory data reset karne se aapke phone mein mauzood sabhi data delete ho jayenge. Isse pahle apne phone ka backup jaroor le le. Android Smartphone mein setting mein Backup & reset ke ander Factory data reset ka option milega. Isse phone mein mauzood unchaha data delete ho jayega aur aapke smartphone ka performance behtar ho jayega.  • Memory card mein application ko kare move aur install :- Aapke smartphone mein agar internal memory kam hain toh koshish kare ki memory card mein hi application aur game ko store kare. Vahi phone mein dher saare application pahle se install hain toh kuch ko aap memory card mein move kar sakte hain. Phone ki setting mein Application Manager mein yeh option aapko dikhayi dega. Yaad rahe Phone ke hang hone ki sabse badi wajah internal memory ka bharna hota hain.
