Agar aap Smatphone user hain to, in dino whatsapp aapke liye jaroori ho gya hain. Jis tarah se whatsapp users ki sankhya badhti ja rahi hain. Aise mein chatting karna aur bhi asaan banta ja raha hain. Halaki iss Messaging App mein bahut se features aise hain, jinhe janna aapke liye jaroori hain. To aayiye jane aise hi khas features. 1. Disable Read Receipts/Last Seen Timestamps Whatsapp mein messaging ke dauraan aapne dekha hoga ki jise aap message karte hain, vo message 2 tick ke saath pahuchta hain. Iska matlab yeh hota hain, ki message server par pahuch gya hain. Aur jab yeh tick blue color mein ho jate hain to vah receiver dwara padh liya jata hain. Halaki ki agar aap Blue tick ko hatana chahte hain , to aapko  Settings > Account > Privacy mein jakar Read Receipts ko Disable karna hoga. Vahi agar aap chahte hain ki koi bhi aapka last seen na dekh sake to ise aap off bhi kar sakte hain.  2. Change Number Keep Whatsapp Account Kisi user ko apna whatsapp number ko change karne ka man kar raha hain, to yeh bahut hi asaan hain. Aap setting mein jakar purane number ko change kar sakte hain. Halaki iss process mein aapka Whatsapp account delete nahi hoga. 3. Check if Whatsapp servers are down Aksar aapne dekha hoga ki, Chatting karte-karte kai message  ruk jaate hain. Halaki ki yeh aamtaur par net service par depend karta hain. Agar aapke phone ka network theek se work nahi kar raha hain, to aisi situation ka samna karna pad sakta hain. Lekin kabhi-kabhi aisa bhi hota hain ki Whatsapp server hi sahi se work nahi kar raha hota hain. Ab isko janne ke liye Settings mein jakar System Status check karna hota hain. 4. Chat Backups on Google Drive Koi Smartphone user kisi bhi chat ko miss nahi karna chahta hain. To iske liye sabse best option hain, Chat Backups on Google Drive. Yani ki aap koi bhi chat history ko Google Drive mein save kar sakte hain aur yeh aapki backup file ki tarah Google Drive mein pada rahega. 5. Pay for a Friend Agar aap apne dost ka payment karna chahte hain, to aapke liye Pay for a Friend ka feature available hain. Jiske jariye aap contact list mein jakar jiska payment karna ho, usay select karke asaani se kar sakte hain. Halaki ki isme 1 Saal, 3 Saal aur 5 Saal ka option uplabadh hoga.
