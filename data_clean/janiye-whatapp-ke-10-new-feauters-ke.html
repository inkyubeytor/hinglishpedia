WhatsApp ne duniya par apna raaj kayam kar liya hain. WhatsApp ke success hone ka secret yeh hain ki yeh time to time apne features update karta rahta hain. Veh hamesha hi apne users ke liye naye-naye features add karta rahta hain. Aise mein veh apne is process mein 10 naye feautes phir se add karne wala hain. Aaiye jante hain veh kaun se new features hain jo Whatsapp jinhe Add ya Update karne wala hain. WhatsApp ke 10 New features in Hindi/Urdu :-  • End encryption  :-  WhatsApp ab send-receiver dono ke beech se calls aur message se encryption feature ko khatm kar raha hain. Iska matlab yeh hain ki ab koi bhi third person kisi ke message nahi padh sakega. • Video Calling :- Ab  jald hi Whatsapp mein Video Calling App bhi shamil hoga. Video Calling  feature Google Play Beta Testing program ke under test hua hain. Ab yeh bahut hi jald Android users ko milne wala hain. • Music Sharing :- WhatsApp mein jald hi users ko Music sharing feature bhi mil jayega. Users is feature  se Apple Music store aur iski device mein Stored music ko easily share kar sakenge. • Big Emoji  :- Abhi bhi Whatsapp mein Emoji hain. Users inka bharpoor istemal karte hain, Lekin jald hi isme bade changes hone wale hain. Naye features update hone ke baad users ko Big size wale Emoji milenge. • Quotation :- Whatsapp ke naye feature mein quotation wala feature bhi shamil hoga. Update karne ke baad jab users kisi ko selected word mein reply karna chahega, tab in quotation ko use kar sakta hain. • Call Back ;- WhatsApp bar jald hi Call Back ka option bhi milega. Abhi company ne Android Beta version par yeh feature roll out kiya hain, Lekin jald hi yeh Google play Store par bhi roll out  ho jayega. • Voicemail :- Yeh feature tab aur zyada helpful hoga jab koi aapki call pick karne ke baad cut karde. Is dauran aap usay voicemail bhi send karke apna reason bata sakte hain ki aapne call kyon ki thi. • Mention & Group invite :- Mention yani ki Tag feature Facebook jaisa kaam karega. Mention user ka naam colored text mein hoga. Isse ek user dusre ko link bhej kar group join karne ke liye bhi invite kar sakega. • Gif support :- WhatsApp jald hi gif support feature add karega. Iske add hone ke baad user ko kafi accha feel hoga. Abhi company ise iOS Beta Version 2.16.7.1 par test kar rahi hain. Aapki jankari ke liye bata de Gif images vah photo hote hain jo chalte firte ya hilte-doolte hain ya phir jinhe live wallpaper bhi kaha jata hain. • New Font :- Naye font ka naam Fixedsys rakha gya hain. Yeh pahle font se thoda patla hain. FixedSys font paane ke liye kisi bhi font mein mauzood content ke start aur end mein teen quotes lagane honge.
