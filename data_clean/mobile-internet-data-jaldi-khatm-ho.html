Aap apni jaroorat ke hisaab se mobile internet data pack khareedte hain, lekin kuch hi dino mein aap paate hain ki saara ka saara mobile data khatm ho gya, jabki aapne uska istemal nahi kiya tha, phir bhi khatm ho gya. Aakhir aisa kyon hua? Aaj jyadatar users ke dimaag mein isi tarah ki baatein aati rahti hain aur phir aap network operator ko dosh dekar apne aap ko shant kar lete hain. Iske baad aap naye seere se apne Smartphone mein naya data pack dalwate hain, agar aap thoda sa dhyaan de toh khud pata chal jayega ki aakhir mein mobile data itni jaldi khatm kyon ho jata hain ya phir aap bahut saare internet mobile data ko kaise bacha sakte hain. Aaj hum aapko internet mobile data bachane ke kuch tips de rahe hain, Inhe follow kare aur bachaye apna internet data :- • Internet Limit set kare :- Jab aap apne smartphone mein internet pack dalwate hain toh uski data limit set kare. Jaise pure mahine mein aapki kitna data kharch karna hain, iski settings data usage mein jakar set kar sakte hain. Aap rojana kitna data usage karte , yeh iski jankari dega. • Possible ho toh na kare web calling :- Aksar log normal call ki jagah web calling karte hain, jabki yeh calling jyada mehngi padti hain. Isliye behtar hoga ki aap normal call hi kare aur jaha tak sambhav ho sake web calling karne se bache. Agar aapko international call karni hain toh aap web calling ka fayda utha sakte hain. • Photo ya video upload karne se bache :- Smartphone ka data sabse jyada videos aur photo ki uploading aur downloading mein kharch hota hain. Koshish kare ki phone mein kam se kam video aur photo download aur upload kare. • Unwanted notification ko band kare :- Bahut se apps kai tarah ke notifications bhejte rahte hain. Isliye jaha tak ho sake inhe band karke rakhe. Aisa karne ke liye application ki settings mein jakar push notification ko off karde. • Mobile data bachane ke liye istemal kare data saver :- Aajkal takareeban sabhi Smartphone mein Data saver ka option mauzood hota hain. Isse aap apne data ki khapat par nazar rakhkar usay bacha sakenge. Data saver ka option aapko setting mein milega. Aap ise browser se enable kar de. Yeh option UCI Browser, Chrome aur Opera samet sabhi mein mauzood hota hain.  • Light version :- Bahut se aise apps hain jinka light version hain. Yeh aap kam data ki khapat karte hain. Jaise ki Facebook Light aur Facebook Messenger light aadi. Inke istemal se kafi saara data bach jata hain. • Background data ko kare restrict :- Aaapke smartphone mein aise apps jo background mein run karte hain. Yeh apps bhari matra mein data ka istemal karte hain. Tab aap inhe restrict kar sakte hain. Iske liye background data restrict ke option ko choone yeh aapko settings ke ander Data usage mein milega. Iske liye aapko top par 3 dot dikhenge, jinhe click karenge toh yeh option mil jayega.
