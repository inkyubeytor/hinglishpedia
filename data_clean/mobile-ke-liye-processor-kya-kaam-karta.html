Mobile ke liye Processor kya hain? Mobile phone hi nahi, balki ab unke buyers bhi smart ho rahe hain. Phone ab bas Camera, Operating System ya screen size dekh kar nahi khareeda jata, Micro-Processor kaisa hain, Kitne Core ka hain, Uski Speed kya hain, Jaise swaal log puchne lage hain. Hum aapko bata rahe hain processor ke baare mein :-  Mobile ke liye kya hain Processor :- Suppose kijiye Mobile agar insaan hota, toh Processor uska dimaag hota. Apne smartphone par aap jo bhi command dete hain, Processor oon par amal karta hain. Processor jitna tez hoga, Utni tez hi multitasking, gaming, Photo aur video edit honge. Ise Central processing unit yani CPU kahte hain jo ek electronic circuit hota hain. Inki performance ko clock rate ki speed se napa jata hain, Yani koi performance per second se cycle se ki gyi, Yeh Hertz (Hz), Kilohertz kHz, Megahertz MHz, Gigahertz GHz, aur Terahertz THz ke scale par naape jate hain. Aamtaur par 1 GHz se lekar 2.4 GHz tak ke processor mobile phone mein mil rahe hain.  Core ki kahani :- iPhone 4 tak hum Single Core Processor hi dekhte thay, Yani ek hi Processor core unit hoti thi. Inki speed mein koi shak nahi tha, Lekin jis tarah phone par jyada se jyada multi-tasking features aa rahe hain, Ab 8 Core wale processor tak ban rahe hain. Ise aur asaani se samajhte hain.  Dual Core Processor ko samajh lijiye ki veh 2 dimaag wala processor hain, Jisme 2 CPU Core ko ek processor mein rakha gya hain. Ab aap phone par jitni jyada command denge, dono processor milkar usay anjaam denge. Jo Single Core se jyada fast honge aur kam battery bhi kharch karenge.  CPU ka size toh companies ek limit tak hi badha sakti hain, Kyonki inke aage heating jaise masle aa sakte hain, Lekin usay ek se jyada core dekar performance next level tak le ja sakti hain.  Dual Core ke baad hame Quad Core (4), Hexa-Core (6) aur Octa-Core (8) Processor dekhne ko milte hain. Ab toh Quad-Core bhi aam feature hota ja raha hain.  Core ki karigri :- Jyada Core hone se Smartphone ki performance kaise sudharti hain, Ise samjhne ke liye ek Dual Core processor ka example lete hain. Example ke taur par aap travel kar rahe hain aur Smartphone ke Navigation aur Music, dono features ka istemal kar rahe hain. Aise mein Operating system Navigation ka kaam ek CPU ko aur Music streaming ka kaam dusre CPU ko de sakta hain. Yani ki ek-ek task dono ke beech bant (Divide) ho jata hain. Isse dono Core apni puri speed mein jaaye bina task pura kar sakte hain. Puri Speed mein naa jane se frequency bhi kam rahti hain aur phone  ki battery kam kharch hoti hain, performance toh sudharti hi hain.  Jyada Core hone ke fayde :- 1) Jyada tez gaming :- Jab aap koi speed ya action game khelte hain jo multi-core processor ise smooth banata hain. Game kanhi atakata nahi hain aur 3D effect ka pura maza milta hain. Gaming console ki takkar ka experience ap mobile mein multi-core processor se aaya hain. 2) Multi-tasking :- Ek saath kai features par kaam karne ke liye Phone ki speed acchi honi chahiye. Mutli-Tab Internet surfing, Multi-Window, Popup Video jaise feature bina multi-core processor ke soche bhi nahi ja sakte hain. Internet ki speed badhne ke saath phone ki speed ko inhone hi tez kiya hain.  3) Full HD video :- Multi-Core Processor se aapko Phone ki screen par 1080 Pixels tak ka video record ya play milta hain. Ab toh Ultra HD playback ko handle karne ki capacity wale Mobile processor aa chuke hain. Big screen par log behtar video playback chahte hain. Isi tarah mobile phone mein jyada damdaar hote Camera ko bhi processor se accha support milta hain.  4) Battery Life :- Screen badi ho rahi hain, Chamak badh rahi hain, Lekin battery ki apni kuch limitation hain. Kam se kam 1 din pura chalne wali battery ab acchi mani jati hain. Jyada Core hone se Mobile ki battery par bhi accha effect hota hain. Sabhi Core mein kaam divide ho jate hain aur jab jis processor ki jaroorat na ho, uske rook jane se battery ki bachat hoti hain. Jab jyada ki jaroorat hoti hain, toh yeh kisi ek par bojh nahi padne deti. Isliye Dosto Single Core bole toh 1 dimaag wala processor, Dual Core bole toh 2 dimaag wala processor, Quad Code bole toh 4 dimaag wala processor, Hexa Core bole toh 6 dimaag wala processor aur Octa Core bole toh 8 dimaag wala processor hain.  Toh dekha aapne Dual Core, Quad-core, Hexa Core, Octa Core in sabhi mein kya antar ya difference hain. Yani ki Jitne jyada dimaag utna badhiya aur tez kaam karega aapka mobile.
