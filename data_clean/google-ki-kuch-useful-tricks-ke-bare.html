Aaj internet ke zamane mein agar aapko kuch internet par search karna hota hain to sabse pahle aap Google ke paas jate hain. Ji haan aaj ke time mein log Internet par kuch bhi search karne ke liye sabse jyada Google Search engine ka hi istemal karte hain. Aaj hum aapko Google search ke kuch aise tips & Tricks aapko batayenge, jisse aapko kafi fayda hoga. Yeh tricks specially Google search box ke liye. Aaiye jante hain Google ke kuch secret tricks ke baare mein, jinhe shayad hi aap jante hain. Google search ke secret tricks & tips in Hindi/Urdu. :-  • Kisi bhi word ka definition janne ke liye Google Search mein word se pahle Definition type kare. Jaise ki aap Money ki Definition janna chahte hain to Google mein Type kare Definition Money isse aapko Money ka matlab puri tarah se google aapko batayega. • Kisi bhi Flight ki Arrival time aur date check karne ke liye Google Search mein Airline ke naam ke baad Flight number type kare. Example ke taur par aap Air India Airlines ki 143 Flight ki arrival time aur date pata karna chahte hain toh Google Search box mein type kare Air India Airlines 143 aur itna hi type karte hi Google aapko show karega is flight ke arrival hone ka time aur date. • Google Search Box ko aap Calculator ki tarah bhi use kar sakte hain. Iske liye aap Search box mein type kare Calculator aur aapke saamne Calculator aa jayega. Vaise direct Google Search box mein Calculate karne se bhi yeh sahi result show karta hain. Jaisa hi 4-4+2= type karte hi google aapko iska right answer 2 de deta hain. • Google search box mein aap Timer bhi set kar sakte hain. Timer set karne ke liye Google search box mein type kare Set Timer __ hours/minutes aur yeh aapke liye timer set kar dega. Example ke taur par mujhe 30 minutes ka timer set karna hain to main Google search box mein type karoonga Set Timer 30 minutes iske baad google mere liye timer set kar dega. • Kisi bhi English word ka apni language mein translate karne ke liye google search box ka bhi aap istemal kar sakte hain. Example ke taur par main Settlement word ka meaning Hindi mein janna chahta hoon to main Google search box mein type karoonga Settlement translation in Hindi ya phir main yeh bhi type kar sakta hoon Settlement in Hindi in 2 dono tarah se type karne par google ek translation ka kaam karne lagta hain. Aur yeh mere saamne Settlement ka meaning Hindi mein show karne lagta hain. Isi tarah aap kisi bhi language mein kisi bhi word ka translate Google se karwa sakte hain. Jaise ki main Optics ka matlab Urdu mein janna chahta hoon toh main Google Search box mein type karoonga Optics in Urdu toh Google automatically iska Urdu mein matlab aapko batayenga. • Local Area ke baare mein kuch bhi search karna chahte hain to Google Maps mein jakar Type kare. Example ke taur par Google Maps mein type kare Restaurants to yeh aas-paas ke sabhi restaurants ki list aapke saamne show karne lagega. • Kisi specific format mein file search karna chahte hain to File name ke aage Filetype: type kare. Example ke taur par main Health Tips ki file PDF format mein search karna chahta hoon toh main Google search box mein type karoonga Health Tips Filetype:PDF iske baad Google mujhe pdf wali sabhi pages show karne lagega. • Google search par aap asaani se stock price, chart aur isse related sabhi information aap paa sakte hain. Jaise ki aap Dabur ke share janna chahte hain to type kare stock:dabur itna hi type karne par Google aapke saamne Dabur company ke share, price sabhi kuch show karne lagega. • Apne City ke weather janne ke liye bhi aap Google search box ka istemal kar sakte hain. Google search mein Weather type karne par baad City ya place ka naam type kare aur result aapke saamne hoga. Example ke taur par main Jalandhar ka weather report janna chahta hoon to maine Google mein type kiya Weather Jalandhar iske baad Google ne mujhe Jalandhar city ka weather report show kiya. • Currency converter ke roop mein bhi Google search box aapki madad karta hain. Jaise ki main 1$ ko Indian Rupees mein convert karna chahta toh main Google par typer karoonga 1$=inr toh Google automatically 1$ ki value Indian rupees mein show karne lagega. Iske alawa agar main Kisi dusri countries ki currency ko bhi google search mein convert kar sakta hoon. Example ke taur par maine 500 Indian rupees ko pound mein convert karne ke liye type kiya 500 Inr to Pound toh Google ne 500 Indian rupees ki value British pound mein show ki. Yani ki Google search box ko aap Currency converter ki tarah bhi use kar sakte hain. Yeh har country ke money ko aapas mein convert karke uski value bata deta hain. • Agar aap janna chahte hain ki aapke City ke kisi Cinema/Multiplex mein kaun si movie abhi lagi hain toh aapko type karna hoga Movies in apne shahar ka naam. Example ke taur par main yeh janna chahta hoon ki Ludhiana mein is samay kaun si movies kaun se theater mein lagi huyi hain toh main Google search box mein type karoonga Movies in Ludhiana bas itna hi type karte hi Google mujhe Ludhiana mein lagi sabhi movies ke naam aur timing kis jagah yani ki cinema house mein lagi hain sabhi information show karne lagega. • Google search box se aap apne city ka sunrise aur sunset hone ka exact time jaan sakte hain. Jaise ki main janna chahta hoon ki Jalandhar mein sunrise kab hoga to maine type kiya Sunrise in Jalandhar  to Google ne mujhe bata diya ki kal kitne baje Jalandhar mein suraj nikalega. Isi tarah sunset hone ka bhi time maine google mein search kiya aur type kiya Sunset in Jalandhar to isne mujhe suraj ke doobne ka time bhi show kiya. Toh aap Google se apne shahar ya kisi bhi jagah ka sunset/sunrise hone ke time ke baare mein asani se jaan sakte hain. • Google search box mein Zerg Rush  type karte hi aap Zerg Rush ki game khelne lagte hain. • Google search box mein aap asani se apne favorite serials ke episodes bhi search kar sakte hain. Jaise hi maine google search box mein Type kiya Saath Nibhaana Sathiya Episodes   toh main hairaan rah gya. • Google search box mein aap Cricket score bhi jaan sakte hain. Bas aapko Type karna hain Cricket Score toh yeh aapke saamne Live sabhi cricket matches ke score show karne lagega. Cricket se judi sabhi information aap google se jaan sakte hain. Jaise main janna chahta hoon Cricket schedule to yeh meri country ki cricket team ke sabhi schedule show karne lagta hain. Isi tarah aap kisi bhi Cricket team ki schedule ke baare mein Google se direct jankari le sakte hain. Jaise ki England ki cricket team ke schedule janne ke liye maine type kiya England Cricket Schedule to mere saamne England team se judi sabhi schedule show hone lage. • Aap Google se apne favorite singer ke songs bhi jaan sake hain.
